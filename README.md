実務志向の匿名掲示板アプリ — 設計から実装まで独自に拡張  
教材の範囲にとどまらず、ログ設計・セキュリティ・i18n など実務を意識した独自機能を追加し、アプリケーションとして発展させました。


本番環境で公開中:  https://junya-code.com

実装内容

インフラ・デプロイ（AWS）
- AWS EC2（Amazon Linux 2023）上に本番環境を構築
- Nginx によるリバースプロキシ設定
- Certbot による SSL/TLS（HTTPS）化
- 環境変数による機密情報管理（メール送信設定など）
- Docker,CI/CD運用
- アプリログの監視・運用


投稿（スレッド・コメント）
- スレッド作成 / 編集 / 削除
- コメント投稿 / 削除
- 文字数制限・バリデーション
- ページネーション対応


ユーザー管理
- ユーザー登録
- ログイン／ログアウト
- 認証・認可（Spring Security）
- 権限管理（管理者／一般ユーザー）


表示・検索
- スレッド一覧表示（ページネーション）
- コメント一覧表示（ページネーション）
- キーワード検索
- ソート機能
- フラッシュメッセージ
- レスポンシブデザイン対応


リアクション
- いいね機能（非同期処理）


セキュリティ・安全対策
- Spring Security によるアクセス制御
- ブラウザセッション UUID によるスパム防止（IP 依存なし）
- 投稿間隔制限（クールダウン方式）
- UUID + IP + User-Agent を組み合わせた監査ログ
- 不正アクセス検知
- CSRF 対策
- 入力バリデーション


バックエンド内部処理
- DB CRUD
- トランザクション管理
- Filter によるブラウザセッション管理
- 責務分離されたログ保存（PAGE_VIEW / CREATE_POST / LOGIN_SUCCESS など）
- エラーハンドリング（GlobalExceptionHandler）
- 一部機能の単体・統合テスト


通知
- お問い合わせ送信時の自動メール送信
- ユーザーへの自動返信
- 管理者への通知メール


管理者向け機能
- 管理者専用 UI
- ユーザー削除
- 特定ユーザーの投稿一括削除
- ログの保持期間管理（自動削除）
